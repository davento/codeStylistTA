# List of Guideline References Used

The following lists the references used for the guidelines per programming language this app considers.

## Python
- Guido van Rossum, Barry Warsaw, Alyssa Coghlan (2001). PEP 8 – Style Guide for Python Code [https://peps.python.org/pep-0008/](https://peps.python.org/pep-0008/)

- Jasmine Finer (2024). How to Write Beautiful Python Code With PEP 8 [https://realpython.com/python-pep8/](https://realpython.com/python-pep8/)

- GeekForGeeks (2020). PEP 8 : Coding Style guide in Python [https://www.geeksforgeeks.org/pep-8-coding-style-guide-python/](https://www.geeksforgeeks.org/pep-8-coding-style-guide-python/)

## C++
- Bjarne Stroustrup, Herb Sutter (2024). C++ Core Guidelines [https://isocpp.github.io/CppCoreGuidelines/CppCoreGuidelines](https://isocpp.github.io/CppCoreGuidelines/CppCoreGuidelines)

- (2024) [https://chatgpt.com/share/67803034-cc04-800a-885b-22eb85f58481](https://chatgpt.com/share/67803034-cc04-800a-885b-22eb85f58481) with some edits


## JavaScript
- Mozilla Developer Network Web Docs contributors, latest update by Joshua Chen (2024). Guidelines for writing JavaScript code examples [https://developer.mozilla.org/en-US/docs/MDN/Writing_guidelines/Writing_style_guide/Code_style_guide/JavaScript](https://developer.mozilla.org/en-US/docs/MDN/Writing_guidelines/Writing_style_guide/Code_style_guide/JavaScript)

- [Aboukhadijeh, F. et al](https://standardjs.com/authors) JavaScript Standar Style. [https://standardjs.com/rules](https://standardjs.com/rules)

- CS 180 Java Programming Standards [https://www.cs.purdue.edu/homes/cs180/java_programming_standards.html](https://www.cs.purdue.edu/homes/cs180/java_programming_standards.html)

- Oracle (1999). Code Conventions for the Java TM Programming Language [https://www.oracle.com/docs/tech/java/codeconventions.pdf](https://www.oracle.com/docs/tech/java/codeconventions.pdf)
